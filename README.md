# AI Handler – Extension MakeCode (DaDa:bit + WonderCam)

Cette extension fournit des blocs simples pour réaliser le projet **AI Handler** :

- Suivi de ligne (ligne noire sur fond clair)
- Détection de couleur (ID1) par WonderCam
- Approche et prise de l’objet
- Transport jusqu’à la destination
- Dépôt de l’objet
- Demi-tour et reprise du suivi de ligne

## Dépendances
- DaDa:bit (Hiwonder)
- WonderCam (Hiwonder)

## Exemple d’utilisation dans MakeCode

1) Au démarrage :
- `AI Handler → initialiser AI Handler (couleur ID 1)`
- (optionnel) `AI Handler → régler paramètres ...`
- (optionnel) `AI Handler → régler vitesses ...`

2) Toujours :
- `AI Handler → cycle AI Handler (1 tour)`

C’est tout.
